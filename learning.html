경사 하강법 (Gradient Descent) 이란?

경사 하강법은 머신러닝 모델 학습의 핵심인 최적화 알고리즘입니다.
모델의 예측 오류(손실 함수)를 최소화하기 위해, 손실 함수의 기울기(경사)를 따라 조금씩 이동하며 최적의 가중치(W)와 편향(b) 값을 찾는 방법입니다.

  • 기울기(Gradient): 현재 위치에서 손실 함수가 가장 가파르게 증가하는 방향을 나타냅니다.
  • 학습률(Learning Rate): 한 번 이동할 때 경사를 따라 얼마나 크게 움직일지를 결정하는 값입니다.
    이 값이 너무 크면 최적점을 지나칠 수 있고, 너무 작으면 학습 속도가 느려집니다.

데이터 수집, 정제, 및 전처리 (Preprocessing)

전체 분석 과정에서 가장 많은 시간을 투자해야 하는 단계로, 원시 데이터를 모델링에 적합하도록 클리닝하고 변환하는 핵심 실무 역량을 키웁니다.

목표: 모델링에 적합하도록 원시 데이터를 클리닝 및 변환
데이터 수집 및 관리

  • 외부 데이터 수집: 웹 크롤링 (BeautifulSoup, Selenium) 및 API 연동을 통한 비정형 데이터 수집
데이터 전처리 (Data Preprocessing) 심화
  • 결측치 처리: 단순 대체를 넘어 MICE, KNN Imputation 등 통계적/모델 기반 대체 기법
  • 이상치 탐지: IQR, Z-score, Isolation Forest 등 알고리즘 기반 이상치 식별 및 처리 전략 수립
  • 데이터 변환: 로그 변환 (Log Transformation) 및 제곱근 변환 등 비대칭 분포 데이터의 정규성 확보
  • 스케일링: Min-Max Normalization과 Standard Scaling의 차이점 및 적용 시기 이해
  • 피처 엔지니어링: 도메인 지식을 활용하여 날짜 데이터 분할, 범주형 변수 조합 등 모델 성능 향상을 위한 파생 변수 생성

1. Min-Max 스케일링

데이터를 최솟값(Min)과 최댓값(Max)을 사용하여 특정 범위(보통 0과 1 사이)로 정규화(Normalization)하는 방법입니다.
데이터의 분포 형태는 유지됩니다.

 
  • 특징: 모든 특성 값을 동일한 0과 1 사이의 범위로 변환합니다.
  • 단점: 데이터에 이상치(Outlier)가 있을 경우, 최댓값이나 최솟값이 크게 영향을 받아 데이터가 좁은 범위에 몰리게 될 수 있습니다.
  • 주요 사용처: 신경망의 활성화 함수(Sigmoid, Tanh)처럼 입력 값의 범위가 특정 범위(예: 0~1)로 제한될 때

2. 표준화 (Standardization)

데이터를 평균이 0이 되도록, 그리고 표준편차가 1이 되도록 변환하는 방법입니다.
이는 Z-Score 정규화라고도 불립니다.

 
  • 특징: 데이터의 실제 최솟값/최댓값에 의존하지 않으므로 이상치에 덜 민감합니다. 변환 후에도 특정 범위로 제한되지는 않습니다.
  • 장점: 데이터가 가우시안 분포(정규 분포)를 따를 때 특히 유용하며, 경사 하강법 기반 모델의 수렴 속도를 개선하는 데 효과적입니다.
  • 주요 사용처: 선형 모델(회귀, 로지스틱), 거리 기반 모델(KNN, K-평균), 그리고 이상치의 영향을 줄여야 할 때

피처 엔지니어링 (Feature Engineering)

피처 엔지니어링은 도메인 지식을 활용하여 기존 원시 데이터로부터 모델 학습에 사용할 새롭고 의미 있는 특성(Feature)을 생성하는 과정입니다. 데이터 전처리 중에서도 가장 창의적이며 모델 성능에 결정적인 영향을 미칩니다.

  • 예시: '생년월일' 데이터를 '나이', '요일', '계절' 등으로 분리하거나, 두 변수 '가격'과 '수량'을 곱하여 '총매출'이라는 새로운 특성을 만드는 것.
  • 목표: 모델이 데이터의 패턴을 더 쉽게 찾고 일반화 능력을 높이도록 돕는 것입니다.

탐색적 분석(EDA) 및 시각화 (Visualization)

데이터의 초기 패턴과 이상 징후를 파악하고, 분석 과정과 결과를 비즈니스 의사결정권자에게 효과적으로 소통하기 위한 기술을 배웁니다.

목표: 데이터의 숨겨진 패턴을 파악하고 분석 결과를 효과적으로 소통
탐색적 데이터 분석 (EDA)
  • 기술 통계: 그룹별/변수별 평균, 분산, 사분위수 등 계산을 통한 데이터 초기 이해
  • 데이터 관계: 상관관계 분석 (Correlation Matrix), 분산 분석 (ANOVA)을 통한 변수 간의 연관성 검토
  • 가설 설정: EDA 결과를 바탕으로 이후 모델링 방향을 설정할 수 있는 잠정적인 가설 도출
데이터 시각화 및 전달 (Visualization)
  • 기본 시각화: Matplotlib을 이용한 기본 차트 (막대, 선, 산점도) 및 Seaborn을 이용한 통계적 시각화 (Violin, Pair Plot)
  • 고급 시각화: Plotly, Bokeh 등 인터랙티브(대화형) 시각화 도구를 활용하여 사용자가 직접 데이터를 탐색할 수 있는 환경 제공
  • 대시보드: Tableau, Power BI 등의 BI 툴을 이용한 데이터 기반 의사결정 지원 시스템(DSS) 구축
  • 스토리텔링: 분석 결과에 따라 차트 유형을 선정하고, 데이터가 전달하는 메시지를 명확하게 구성하는 능력

데이터 스토리텔링 (Data Storytelling)

데이터 스토리텔링은 단순한 수치나 그래프를 넘어, 분석 결과를 청중이 이해하기 쉬운 서사(Narrative)로 구성하고 전달하는 능력입니다. 효과적인 데이터 스토리텔링은 듣는 사람의 행동 변화를 유도하는 핵심 기술입니다.

  • 구성 요소: 데이터(정확성), 시각화(명료성), 서사(맥락과 통찰)
  • 목표: "무엇을 봤는지" 뿐만 아니라, "그래서 무엇을 해야 하는지"에 대한 결론과 행동 방안을 제시합니다.

머신러닝 및 모델링 (Machine Learning)

데이터를 기반으로 미래를 예측하거나 패턴을 식별하는 핵심 알고리즘을 학습하고, 실제 문제 해결을 위한 모델 구축, 평가 및 최적화 능력을 배양합니다.

목표: 데이터로부터 학습하여 정확한 예측 및 패턴 식별 모델 구축
핵심 머신러닝 알고리즘
  • 지도 학습 (Supervised): 로지스틱 회귀 (이진 분류), 선형 회귀 (연속 값 예측)
  • 분류 알고리즘: 결정 트리, 랜덤 포레스트 (앙상블), 서포트 벡터 머신 (SVM)의 원리 이해 및 적용
  • 비지도 학습 (Unsupervised): K-Means/DBSCAN (군집화), PCA (주성분 분석을 통한 차원 축소)
모델 평가 및 최적화
  • 모델 훈련/테스트: 학습 데이터와 테스트 데이터 분리, 과적합/과소적합 방지 전략 (규제 L1, L2)
  • 평가 지표: Confusion Matrix (혼동 행렬), ROC-AUC 커브, 정밀도/재현율 트레이드오프 이해
  • 앙상블 기법: 배깅(Bagging), 부스팅(Boosting)의 차이 이해 및 XGBoost, LightGBM 등 실전 도구 활용
  • 주요 도구: Scikit-learn의 모든 모듈 (Preprocessing, Model, Evaluation) 숙련

앙상블 기법 (Ensemble Methods)

앙상블 기법은 여러 개의 약한 학습기(Weak Learners)를 결합하여 하나의 강력한 예측 모델을 만드는 방법입니다. 단일 모델보다 훨씬 높은 정확도와 안정성을 제공하여 실무에서 광범위하게 사용됩니다.

  • 배깅(Bagging, 예: 랜덤 포레스트): 독립적인 다수 모델을 학습시킨 후 결과를 평균/투표하여 최종 예측을 수행. 과적합 방지에 효과적
  • 부스팅(Boosting, 예: XGBoost): 이전 모델의 오류를 다음 모델이 보완하며 순차적으로 학습. 일반적으로 가장 높은 예측 성능을 보입니다.


  XG Boost (Extreme Gradient Boosting):
: 대용량 분산 처리를 위한 그래디언트 부스팅으로 의사결정나무에 Boosting기법을 적용한 Ensemble 알고리즘

딥러닝 및 실무 응용 (Deep Learning & Advanced Application)

이미지, 텍스트와 같은 비정형 데이터를 처리하고, 대규모 분석을 위한 빅데이터 플랫폼 및 실제 서비스 배포를 위한 운영 환경을 이해하여 실무 역량을 완성합니다.

Transformer 아키텍처

Transformer는 2017년 Google에서 제안한 모델 구조로, 기존 NLP의 RNN/LSTM을 대체하며 현재 GPT, BERT 등 대형 언어 모델(LLM)의 근간이 되었습니다.

  • 핵심: Attention(어텐션) 메커니즘만으로 인코더와 디코더를 구성하여 시퀀스 데이터를 병렬 처리할 수 있게 합니다.
  • 장점: RNN의 순차적 처리 한계를 극복하여 학습 속도가 빠르고 장기 의존성 문제를 해결하여 성능이 혁신적으로 향상되었습니다.
