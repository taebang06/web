학습 자원 플랫폼

수업 외 자율 학습을 위한 필수 온라인 플랫폼과 개발 환경을 안내합니다.

온라인 학습 플랫폼

Kaggle (캐글)

전 세계 데이터 과학자들이 참여하는 데이터 분석 및 머신러닝 경진대회 플랫폼. 실전 데이터를 다루고 다른 사람의 코드를 학습할 수 있는 최고의 환경입니다.

Coursera / edX

세계 유수 대학 및 기업의 데이터사이언스 MOOC (온라인 공개 강좌)를 수강할 수 있는 플랫폼. 부족한 이론이나 특정 기술을 보완하는 데 유용합니다.

GitHub (깃허브)

버전 관리 시스템의 필수 도구. 모든 코드 프로젝트를 체계적으로 관리하고, 포트폴리오로 활용할 수 있습니다. 1학년 때부터 사용에 익숙해져야 합니다.

주요 개발 환경 (IDE)

주피터 노트북 (Jupyter Notebook)

웹 브라우저 기반의 대화형 컴퓨팅 환경으로, 코드, 실행 결과, 설명 텍스트, 시각화를 하나의 문서(.ipynb)에 결합하여 작성하고 공유할 수 있게 해줍니다.


셀(Cell) 단위 실행이 가능하여 데이터 분석 과정이나 실험적인 코드를 순서대로 기록하고 결과를 즉시 확인할 때 가장 효과적입니다.

구글 코랩 (Google Colab, Colaboratory)

주피터 노트북 환경을 Google 클라우드에서 제공하는 서비스입니다.


별도의 설치나 환경 설정 없이 무료로 고성능 GPU/TPU 자원을 사용할 수 있습니다. Google Drive와 연동되어 코드와 데이터를 쉽게 공유할 수 있습니다.

파이참 (PyCharm)

JetBrains에서 개발한 Python 전문 통합 개발 환경 (IDE)입니다.


강력한 코드 자동 완성, 리팩토링, 디버깅 기능 등 소프트웨어 개발에 필요한 모든 전문적인 기능을 제공합니다.

VS 코드 (Visual Studio Code, VS Code)

Microsoft에서 개발한 무료 오픈소스 코드 편집기입니다.


가볍고 빠르면서도 확장 기능(Extension)을 통해 다양한 언어(Python, R, JavaScript 등)와 개발 환경을 지원합니다.
